# 任务完成检查清单

## 完成任务后应执行的步骤

### 1. 类型检查
```bash
npx tsc --noEmit
```
确保 TypeScript 编译无错误。

### 2. 构建测试
```bash
npm run build
```
确保项目能够成功构建。

### 3. 开发测试
```bash
npm run dev
```
启动开发服务器，手动测试修改的功能。

### 4. 代码审查
- 确保代码风格与项目一致
- 检查中文注释是否清晰
- 确保类型定义完整

## 注意事项
- 项目没有自动化测试，需要手动测试
- 项目没有 linter/formatter，代码格式需保持一致
- 修改 Electron 主进程代码后需要重启开发服务器
